OpenStdin
StdinOnce Close stdin when client disconnects
EnvEnvironment variables to set in the container. Example ORACLE_SID=ASF and if multiple separate with comma e.g ORACLE_BASE=/u01/app/oracle,ORACLE_SID=ASF

Healthcheck A test to perform to check that the container is healthy. Example:

"/u01/app/database":{}
}
WorkingDir

NetworkDisabled
MacAddress
